# Proposal Case Study — Proof Generator

Generates persuasive case studies in Challenge → Solution → Results format. Invoked by `/proposal case-study <result>`. Output: `PROPOSAL-CASESTUDY.md`.

## Steps
For each case study (produce 1-3):
- **Header:** Client name (or "Leading [Industry] Company" if confidential) · Industry · Size · one-line challenge
- **The Challenge:** 2-3 sentences mirroring the prospect's situation
- **The Solution:** 2-3 sentences, approach-focused
- **The Results:** ≥3 specific metrics, before → after with % improvement, plus timeline
- **Client Quote:** one-sentence testimonial if available

## Rules
Match the prospect's industry/size/challenge as closely as possible. Every case study needs ≥3 measurable results and a timeline. If using representative examples, label them clearly. Vague results ("improved efficiency") are worthless — use numbers.
