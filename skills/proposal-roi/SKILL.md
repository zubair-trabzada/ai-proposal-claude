# Proposal ROI — Business Case & ROI Projection

Builds the ROI projection and cost-of-inaction case. Invoked by `/proposal roi <client>`. Output: `PROPOSAL-ROI-[Client].md`.

## Steps
1. Identify the value lever: service → metric → revenue (GEO/SEO → traffic → leads → revenue; reputation → rating → conversion → revenue; etc.). Use the client's business model.
2. Quantify the cost of inaction: ongoing monthly/annual loss from the status quo, tied to audit weaknesses.
3. Build a current-vs-projected table across conservative / moderate / aggressive, with annual value, investment, net ROI, ROI %.
4. State every assumption with its basis.

## Rules
Conservative case must still be positive ROI — if not, flag the scope/price. Transparent assumptions only. Use the client's own metrics where possible.
