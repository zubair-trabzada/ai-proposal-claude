# Proposal Upsell — Expansion Proposal for Existing Clients

Builds an expansion/upsell proposal for a client you already serve. Invoked by `/proposal upsell <client>`. Output: `PROPOSAL-UPSELL-[Client].md`.

## Steps
1. Reference results already delivered (the proof you've earned). Pull from prior reports if present.
2. Identify the next logical service to add (e.g. GEO client → add reputation management or content retainer).
3. Frame the expansion around momentum: "Here's what we achieved together; here's the next ceiling we can break."
4. Price the add-on (often a bump to the next tier or a new line item). Show incremental ROI.
5. Make it easy: a short upgrade addendum, not a full re-pitch.

## Rules
Lead with results delivered, not a cold pitch. Existing clients convert on momentum and trust. Keep it to 1-2 pages. Include the incremental ROI of the add-on specifically.
