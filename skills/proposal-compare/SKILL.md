# Proposal Compare — Two-Approach Comparison

Compares two proposal approaches side-by-side. Invoked by `/proposal compare <a> <b>`. Output: `PROPOSAL-COMPARE.md`.

## Use Cases
- Two pricing models (retainer vs project)
- Two scopes (lean audit vs full management)
- Two positioning angles for the same client

## Output
A comparison table across: investment, deliverables, timeline, expected ROI, risk, close probability. Then a clear recommendation on which to lead with and why, plus when to use the other as a fallback.
